# 🧮 Simple Python Calculator

This is a basic command-line calculator built with Python. It takes two numbers and an operator as input, performs the desired arithmetic operation, and displays the result.

---

## 🚀 Features

    - Accepts user input for two numbers and an operator.
    - Supports basic arithmetic operations:
      - Addition (`+`)
      - Subtraction (`-`)
      - Multiplication (`*`)
      - Division (`/`)
    - Handles divide-by-zero errors gracefully.
    - Displays an error message for invalid operators.

---

## 📋 How It Works
    
    1. The user is prompted to enter the first number.
    2. The user enters an operator (`+`, `-`, `*`, or `/`).
    3. The user enters the second number.
    4. Based on the operator, the calculator performs the appropriate operation.
    5. The result is printed to the console.

---

## 🧑‍💻 Example Usage
    Enter your first number: 10
    Enter an operator (+, -, *, /): *
    Enter your second number: 5
    Result: 50.0
