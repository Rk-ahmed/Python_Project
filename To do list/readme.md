# 📝 To-Do List CLI App

A simple Command Line Interface (CLI) To-Do List application written in Python.

This app allows users to:

    - View their list of tasks
    - Add new tasks
    - Remove existing tasks
    - Exit the app safely

The program runs in a loop and interacts with the user via terminal input/output.

---

## 🚀 Features

    - Simple text-based interface
    - Uses Python lists to store tasks
    - Modular functions for better readability
    - Interactive menu system

---

## 🧠 How It Works

1. The app displays a menu with four options:
- View tasks
- Add a task
- Remove a task
- Exit

2. Based on the user's input, it calls the corresponding function.
3. The app continues to run until the user chooses to exit.
